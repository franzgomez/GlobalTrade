# Global Trade & Market Insights - Argentina

## Descripción del Proyecto

Esta aplicación web presenta tres estudios de mercado exhaustivos para la importación y comercialización de productos premium en Argentina:

1. **Lurisia** - Bebidas Premium (agua mineral, sodas artesanales, aperitivos sin alcohol)
2. **Meltz Dubai** - Chocolates Gourmet (barras con knafeh y pistacho)
3. **Augusta** - Confitería Premium Italiana (panettones y dulces de lujo)

## Estructura de Archivos

### Aplicación Web
- `index.html` - Archivo principal de la aplicación (SPA)
- `styles.css` - Estilos CSS responsive y modernos
- `app.js` - JavaScript vanilla para navegación e interactividad

### PDFs de Contactos
- `Lurisia_Contactos_Argentina.pdf` - Listados de clientes, competidores y canales para Lurisia
- `MeltzDubai_Contactos_Argentina.pdf` - Listados específicos para chocolates Meltz Dubai
- `Augusta_Contactos_Argentina.pdf` - Contactos del segmento ultra-premium Augusta

## Características de la Aplicación

### Navegación
- **Single Page Application (SPA)** sin backend
- Header fijo con navegación principal
- Navegación interna por estudio con scroll suave
- Botones de retorno al inicio desde cualquier sección
- Responsive design para desktop, tablet y móvil

### Estructura por Estudio
Cada estudio incluye las siguientes secciones:
1. Resumen Ejecutivo
2. Análisis PESTEL / Macroeconómico
3. Análisis del Sector / Modelo de Porter
4. Análisis Competitivo (Benchmark)
5. Análisis del Consumidor
6. Tendencias y Oportunidades
7. Recomendaciones Estratégicas
8. Clientes Potenciales (con investigación adicional)
9. Canales de Venta (distribuidores y plataformas)

### Sección Comparativa
- Tabla comparativa de las tres oportunidades
- Análisis ejecutivo de diferencias clave
- Recomendaciones sobre cuándo priorizar cada producto
- Factores críticos de éxito comunes

## Recomendaciones Estratégicas

### Priorización por Perfil de Inversor

**Para Capital Limitado (USD 25-35K):**
- Iniciar con **Meltz Dubai** por menor barrera de entrada
- Aprovechar tendencia viral del pistacho
- Target joven digital con alta adopción

**Para Capital Abundante (USD 115K+):**
- **Lurisia** para construcción de prestigio de marca
- Break-even más rápido y menor riesgo
- Canales HORECA premium establecidos

**Para Enfoque Ultra-Premium:**
- **Augusta** para márgenes altos y exclusividad
- Segmento estacional pero leal
- Regalos corporativos B2B

### Estrategia Combinada Recomendada
1. **Año 1:** Meltz Dubai (validación operativa)
2. **Año 2:** Lurisia (construcción de prestigio)
3. **Año 3:** Augusta (extensión premium)

## Instalación y Uso

### Para GitHub Pages / Netlify
1. Subir los archivos `index.html`, `styles.css`, `app.js` a la raíz del repositorio
2. Configurar el dominio personalizado (opcional)
3. La aplicación funcionará automáticamente como SPA

### Para servidor web local
1. Colocar los archivos en el directorio web
2. No requiere configuración de servidor especial
3. Compatible con cualquier servidor HTTP estático

Los archivos PDF están optimizados para:
- Impresión en formato A4
